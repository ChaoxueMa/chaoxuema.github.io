# Project 配置

一个 React App 的目录结构可以这样的

config/
	webpack.base.config.js
	webpack.prod.config.js
	webpack.dev.config.js
public/
  index.html
scripts/
src/
  assets/
  components/
  pages/
  router/
  store/
		store.js
		reducer.js
	style/
	utils/
  index.js
  App.js
package.json

目录

  • config/ —— 存放配置相关的文件
  • public/ —— 存放 html 相关的文件
  • scripts/ —— 存放脚本相关的文件
  • src/ —— 存放业务文件
    • assets/ —— 存放静态资源
    • components/ —— 存放组件
    • pages/ —— 存放页面
    • router/ —— 存放页面路由
    • store/ —— 存放数据
    • style/ —— 存放样式
    • utils/ —— 存放 js 方法

文件

  • config/webpack.base.config.js —— webpack 公共配置
  • config/webpack.prod.config.js —— webpack 生产环境配置
  • config/webpack.dev.config.js —— webpack 开发环境配置
  • public/index.html —— 页面模板
  • src/store/store.js —— Redux store
  • src/store/reducer.js —— Redux reducer
  • src/index.js —— Webpack entry
  • src/App.js —— React entry
  • package.json —— Node 配置
