软件配置的相关内容，内容包括 Windows 配置、输入法配置、Everything 配置、Xmind:Zen 配置、小飞机配置、Chrome 配置。

一、Windows 配置

1. 基本配置

  1. 「控制面板」→「硬件和声音」→「电源选项」→「系统设置」→
    • 「按电源按钮时：睡眠（用电池 / 接通电源）」
    • 「按睡眠按钮时：睡眠（用电池 / 接通电源）」
    • 「关闭盖子时：休眠（用电池 / 接通电源）」
  2. 「控制面板」→「硬件和声音」→「电源选项」→「编辑计划设置」→
    • 「关闭显示器：15 分钟（用电池 / 接通电源）」
    • 「使计算机进入睡眠状态：30 分钟（用电池），从不（接通电源）」
  3. 「显示文件后缀」+「显示隐藏文件」
  4. 「任务栏、桌面把图标清理干净」
  5. 常规目录放到动到 D 盘
  6. 几个修饰键：Ctrl / Shift / Alt / Win / Cmd / Tab / Esc
  7. 平时不关机，盖子一合，进入休眠模式

2. 常用快捷键

  1. 「Win + Desktop」– 展示桌面
  2. 「Win + 方向键」– 移动窗口
  3. 「Alt + Tab」– 切换窗口
  4. 「Win + Tab」– 不怎么常用的切换窗口
  5. 「Win + Ctrl + 方向键」– 切换桌面
  6. 「Ctrl + A / C / V / Z / Y」
  7. 「Ctrl + R」 / 「F5」– 刷新
  8. 「Ctrl + P」– 打印

二、输入法配置

1. 基本配置

我使用的是「搜狗拼音输入法」中的「小鹤双拼」方案

  1. 把状态栏调整成顺手的长度
  2. 「设置」→「常用」→「双拼」→「方案：小鹤双拼」+「√ 双拼下同时使用全拼音」
  3. 「设置」→「外观」→「候选项数：7」
  4. 把联网的功能全部关掉


2. 常用快捷键

  1. 「Ctrl + Shift + Z」– 调出表情和符号
  2. 「Shift + U + ‘拆解字’」– 查不认识的字的读音

三、Everything 配置

1. 基本配置

  1. 「设置」→「界面」→「√ 单机托盘图标打开搜索窗口」
  2. 「设置」→「结果」→「√ 搜索关键词为空时不显示搜索结果」
  3. 「设置」→「结果」→「√ 双击路径列打开目录」

四、Xmind:Zen 配置

1. 常用快捷键

  1. 「Ctrl + /」– 折叠或展开子主题
  2. 「Tab」– 添加子主题
  3. 「Enter」– 之后添加主题
  4. 「Ctrl Enter」– 添加父主题
  5. 「Spacebar」– 编辑主题
  6. 「Shift + Enter（编辑状态下）」– 换行
  7. 「Enter（编辑状态下）」– 结束编辑

五、小飞机 配置

1. 基本操作

  1. 登录官网购买服务器并仔细阅读使用教程
  2. 下载客户端并添加服务器节点（下载配置文件直接导入）
  3. 测试筛选出好用的网速快的节点（不是每一个节点都是好的）
  4. 使用「Chrome」中的插件「SwitchyOmega」管理代理

2. SwitchyOmega 的使用

  • 之前有配置文件的就先导入之前的配置
  • 直接连接 / XX代理 / 自动切换 – 不使用代理 / 使用代理 / 智能使用代理
  • 添加条件 – 添加并保存当前页面的连接方式

六、Chrome 配置

1. 好用的插件

  1. SwitchyOmega – 管理浏览器的代理
  2. uBlock Origin – 广告屏蔽
  3. Evernote Web Clipper – 保存完整网页到印象笔记
  4. Separate Window – 开启小窗播放视频
  5. Death To _blank – 阻止链接在开一个新标签打开
  6. JSON Viewer – 可以直接解析 JSON 文件

2. 调出配置相关的快捷方式

  1. chrome://settings/– 设置
  2. chrome://apps/– 管理应用
  3. chrome://extensions/– 管理插件

3. 常用快键键

  1. 「鼠标中键单机」– 打开或者关闭
  2. 「Ctrl + T」– 新开标签
  3. 「Ctrl + Shift + T」– 撤销关闭
  4. 「Ctrl + 点击」– 在新标签打开
  5. 「Ctrl + W」– 关闭当前标签
  6. 「Ctrl + R」/「F5」 – 刷新
  7. 「Ctrl + L」– 输入网址
  8. 「Ctrl + Page Up / Page Down」– 切换到上/下一个标签
  9. 「Ctrl + Shift + I」/ 「F12」– 打开发者工具
  10. 「Alt + ← / →」– 前进/后退
  11. 「Alt + Enter」– 在新标签打开
  12. 「Shift + Enter」– 在新窗口打开
  13. 「Ctrl + Shift + Delete」– 删除历史浏览数据
  14. 「Ctrl + N」– 新建窗口
  15. 「Ctrl + Shift + N」– 新建无痕标签
  16. 「Ctrl + P」– 打印

4. Ctrl + P 的打印配置

  1. 「边距：无」
  2. 「√ 背影图形」
  3. 「× 页眉和页脚」
